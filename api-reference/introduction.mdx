The Plant Store API lets you retrieve, create, and delete plant records. All requests must be authenticated with a Bearer token passed in the `Authorization` header. This reference covers authentication setup and links to each available endpoint.

## Authentication

Every request to the Plant Store API requires a Bearer token:

```bash
Authorization: Bearer YOUR_TOKEN
```

Pass this header with every API call. Requests without a valid token return a `400` error.

## Base URL

```
http://sandbox.mintlify.com
```

## Making your first request

<CodeGroup>

```bash curl
curl http://sandbox.mintlify.com/plants \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```typescript TypeScript
const response = await fetch('http://sandbox.mintlify.com/plants', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const plants = await response.json();
```

```python Python
import requests

response = requests.get(
  'http://sandbox.mintlify.com/plants',
  headers={'Authorization': 'Bearer YOUR_TOKEN'}
)
plants = response.json()
```

</CodeGroup>

## Error responses

All errors return a JSON body:

```json
{
  "error": 400,
  "message": "Unexpected error occurred"
}
```

<ResponseField name="error" type="integer">
  The HTTP status code.
</ResponseField>

<ResponseField name="message" type="string">
  A human-readable error description.
</ResponseField>
