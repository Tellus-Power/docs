Mintlify is a documentation platform that turns MDX files into polished, production-ready docs sites. Connect your GitHub repo, customize your brand, and your docs are live — no servers to manage, no build pipeline to configure. Everything from navigation to AI integrations is configured through a single `docs.json` file.

## Get started in minutes

<Steps>
  <Step title="Clone your docs repo">
    Create a docs repo from the Mintlify starter kit or connect an existing repository through your [dashboard](https://dashboard.mintlify.com).
  </Step>
  <Step title="Preview locally">
    Install the CLI and run `mint dev` to see live changes as you write.

    ```bash
    npm i -g mint
    mint dev
    ```
  </Step>
  <Step title="Deploy automatically">
    Install the Mintlify GitHub app from your dashboard. Every push to your default branch deploys your docs instantly — no CI/CD setup required.
  </Step>
</Steps>
